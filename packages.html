Packages

Sublime Package Control

A Sublime plugin for easily incorporating other packages into Sublime (without web browser)

Once installed, enter package name into the tool, and Sublime installs it.

To Install:
  1. Open Sublime Text. Open the console by pressing cmd/ctrl + `
  2. Go to packagecontrol.io/installation
  3. Copy the code from the Sublime Text 3 box, and paste it into the console. Hit return to run the code.
  4. Close and reopen Sublime Text Editor.

Emmet

Quickly insert boilerplate code with certain key combinations; create JsPsych experiments faster!

To Install:
  1. cmd/ctrl + shift + p to open the options console.
  2. Type Package Control Install, and select Package Control: Install Package
  3. In the Package Control menu, search Emmet. Click on the highlighted result to install.
  4. Relaunch Sublime.
Try It:

(1) Download emmet.html (2) Open in sublime (3) Follow the instructions in the comments.


Type in an emmet shortcut followed by tab to insert code

  1. ! in a .html file to insert boilerplate html

  2. easily create and nest HTML element tags:

    > for child relation, + for sibling relation, and () to group

    e.g. table>(thead>tr)+(tbody>tr) yields all of the html tags to create a table with a header and body with 1 row each

Terminal

Sets a shortcut to open a new terminal window at the current file

To Install:
  1. Install 'Terminal' using Package Control: Install Package
Try It:

While open/active in a file in Sublime:

  • Mac: Cmd + shift + t
  • Win: Ctrl + shift + t
