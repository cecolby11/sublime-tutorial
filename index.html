Tools

0. Install Sublime Text

Latest is Sublime Text 3 (in beta). Sublime Text 2 is also fine! Download Here

1. Sublime Package Control

A Sublime plugin for easily incorporating other packages into Sublime (without web browser)

Once installed, enter package name into the tool, and Sublime installs it.


To Install:

  1. Open Sublime Text. Open the console by pressing command + `
  2. Go to packagecontrol.io/installation
  3. Copy the code from the Sublime Text 3 box, and paste it into the console. Hit return to run the code.
  4. Close and reopen Sublime Text Editor.

2. Emmet!

Quickly insert boilerplate code with certain key combinations; create JsPsych experiments faster!

To Install:

  1. command + shift + p to open the options menu.
  2. Type Package Control Install, and select Package Control: Install Package
  3. In the Package Control menu, search Emmet. Click on the highlighted result to install.
  4. Relaunch Sublime.

To Use:

  1. ! + tab in a .html file to insert boilerplate html

Next Up: Color Highlighter

This Sublime plugin highlights colors in Sublime Text

In the ChiLDlab we can use Emmet to code JsPsych experiments faster!

  1. Close and reopen Sublime Text Editor.
  2. Hit command + shift + p to open the options menu. Then type Package Control Install, and click the Package Control: Install Package option highlighted.
  3. In the Package Control menu, type Emmet. Click on the highlighted option to install the plugin.
  4. Close and reopen Sublime.
  5. To make sure it's installed, hit command + shift + p to open the options menu again.
  6. If you type Emmet and you see a slew of Emmet options, then congrats!

User Preferences

  1. Go to Sublime Text > preferences > settings > preferences.sublime-settings -- User

  2. Paste/add In: { "color_scheme": "Packages/Color Scheme - Default/Monokai.tmTheme", "font_size": 19.0, "hot_exit": false, "ignored_packages": [ "Vintage" ], "line_padding_bottom": 2, "remember_open_files": false, "save_on_focus_lost": true, "scroll_past_end": true, "tab_size": 2, "translate_tabs_to_spaces": true, "trim_trailing_whitespace_on_save": true, "word_wrap": true }

    • hot_exit: if true, sublime will close unsaved windows without prompting (and the default is true! eek!)
    • line_padding_bottom: a little extra spacing below each line (default is 0px)
    • scroll_past_end: if true, sublime leaves empty space between last line and bottom of the file (mac default: false, windows/linux: true)
    • save_on_focus_lost: automatically saves file every time you click out of its window (this is the best)
    • word_wrap: if true, sublime disables horizontal scrolling (not disabled by default)

  3. For more: Sublime settings documentation
